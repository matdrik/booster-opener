# BOOSTER OPENER

### 1. Установка проекта (модулей)
```
npm install
```

### 2. Запуск frontend части
```
npm run serve
```

### 3. Запуск backend части
```
node server/server
```

### frontend
```
localhost:**8080** или <ваш ip>:**8080**
```

### Сервер
```
localhost:**3000** или <ваш ip>:**3000**
```
